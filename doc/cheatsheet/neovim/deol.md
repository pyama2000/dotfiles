# Cheat sheet for deol.nvim

## ノーマルモード

| key                   | description                               |
|:----------------------|:------------------------------------------|
| <kbd> Ctrl + o </kbd> | deol をフローティングウィンドウで起動する |

## ターミナルモード

| key                   | description            |
|:----------------------|:-----------------------|
| <kbd> Esc </kbd>      | deol から抜ける        |
| <kbd> Ctrl + z </kbd> | 新しく deol を起動する |
